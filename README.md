## 📦 Available Plans

### 🚀 tdias Commerce Lite

| | |
|:---|:---|
| **Audience** | For those just starting out who need speed and affordability, not complexity |

Lite was built to be cost-effective and run at peak efficiency on entry-level infrastructure, without sacrificing any essential functionality. You have everything you need to set up your store, sell your products, and get paid, with no technical team required to go live.

#### ✅ Features Included

- 📋 Product catalog
- 🛒 Optimized checkout

### 💼 tdias Commerce Pro

| | |
|:---|:---|
| **Audience** | For those who have already proven the business works and want to scale without hitting walls |

Pro is the intermediate tier, balancing cost and capability. With more resources and stronger performance, it is built for stores handling higher sales volume. More products, more simultaneous orders, more business logic, more integrations. All of this paired with an advanced analytics dashboard, multi-warehouse inventory management, custom shipping rules, and support for multiple sellers or sales representatives.

#### ✅ Features Included

- 📊 Advanced analytics dashboard
- 🏭 Multi-warehouse inventory management
- 🚚 Custom shipping rules
- 👥 Multi-seller and representative support
- 🔗 Expanded integrations

> **Ideal for:** growing e-commerces, vertical marketplaces, and operations that demand both control and performance at the same time.
